## Notes App
A simple and responsive Notes App built using JavaScript, HTML, and CSS.
This app allows users to create, edit, and delete notes easily.

## Features
 Add new notes
 Edit existing notes
 Delete notes
 Data saved in Local Storage (notes stay after refresh)
 Responsive design

 ## Technologies Used
HTML
CSS3
JavaScript
